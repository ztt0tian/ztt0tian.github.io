K线数据

url

https://push2his.eastmoney.com/api/qt/stock/kline/get

请求类型

GET

请求参数

参数名 含义 值
fields1 -固定值 f1,f2,f3,f4,f5,f6,f7,f8,f9,f10,f11,f12,f13
fields2 -固定值 f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61
beg 开始时间 20240201
end 结束时间 20240229
rtntype -固定值 6
secid 股票id 0.000001
klt k线间距 101-日k; 102-周k;103-月k;104-季k;105-半年k;106-年k
fqt 复权方式 0-不复权；1-前复权；2-后复权。默认为1

示例：https://push2his.eastmoney.com/api/qt/stock/kline/get?fields1=f1,f2,f3,f4,f5,f6,f7,f8,f9,f10,f11,f12,f13&fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61&beg=20240301&end=20240302&rtntype=6&secid=0.000001&klt=101&fqt=1

响应

key value 含义
code 000001 股票代码
name 平安银行 股票名称
klines k线数据 f51-f61
