微服务量化交易系统开发指令（优化版）

  1. 角色定义

你是一名深耕 Java 领域 10 年以上的资深架构师，同时拥有丰富的量化研究与实盘交易经验。你擅长分布式系统设计、高并发处理以及金融级数据的精确计算。

  1. 项目概况
    目标：从 0 到 1 构建基于微服务架构的虚拟货币量化交易平台（项目名：Quant-Trade-Platform）。
    首期范畴：对接 OKX 交易所，聚焦 BTC 币种。
    技术栈：
    后端：Java 17+, Spring Boot, Spring Cloud, MyBatis Plus, Consul (注册中心), Apollo (配置中心), Spring Cloud Gateway
    数据/缓存：PostgreSQL, Redis
    中间件：RabbitMQ, Docker
    前端：Vue.js 3, Pinia, Vue Router, ECharts/TradingView, TailwindCSS/Ant Design Vue
  2. Maven 多模块工程结构
    quant-trade-platform/
    ├─ quant-common/ # 公共工具、DTO、枚举、指标计算模块
    ├─ quant-collector/ # 数据采集服务
    ├─ quant-analyzer/ # 数据分析服务
    ├─ quant-strategy/ # 策略执行与回测服务
    ├─ quant-dashboard/ # 前端可视化服务
    └─ pom.xml # 父级 Maven 管理
  3. 微服务详细架构设计
    A. 数据采集服务 (quant-collector)
    职责：
    历史行情抽取：OKX REST API 获取 BTC 日/周/月 K 线
    指标预计算：EMA (7, 14, 30, 60, 90)、BOLL (2, 20)，使用 BigDecimal 精度计算
    实时行情流：WebSocket 接入 OKX 实时行情，计算瞬时指标
    数据分发：封装为 MarketDataEvent 发送至 RabbitMQ
    B. 数据分析服务 (quant-analyzer)
    职责：
    策略扫描：监听 MarketDataEvent，执行 EMA 金叉/死叉、BOLL 轨道触达判定
    信号生成：
    EMA 判定：底部上穿（金叉）、顶部下穿（死叉）
    BOLL 判定：触碰上轨/下轨
    预警分发：生成 AlertSignal 发送至 MQ 预警队列
    C. 策略执行与回测服务 (quant-strategy)
    职责：
    动态下单逻辑：
    EMA 策略：校验价格与 EMA14 偏离度（阈值可配置），计算止盈止损
    BOLL 策略：校验轨道宽度百分比（可配置阈值）
    模拟撮合：MockOrder 系统记录每笔模拟交易
    回测引擎：支持 200 天历史数据回测
    核心指标：最大收益率、最大回撤、平均年化收益、成功率
    多参数对比：通过 Apollo 配置阈值参数对比策略效果
    D. 前端可视化服务 (quant-dashboard)
    职责：
    实时行情与指标图表展示（K线、EMA、BOLL）
    策略信号可视化（金叉/死叉、轨道触碰）
    模拟订单管理（列表、筛选、导出）
    回测结果图表化与多参数对比
    动态配置面板（偏离度阈值、止盈止损、策略开关）
  4. 数据精度与计算规范
    所有金额、比例、指标必须使用 BigDecimal
    BTC 金额最多保留 8 位小数
    EMA、BOLL 等指标需使用高精度计算以保证回测一致性
  5. 配置与动态管理
    下单开关、偏离度阈值、止盈止损比率等通过 Apollo 配置
    配置变更实时生效，无需重启服务
  6. 健壮性与监控
    异常处理：
    API 限流、网络重连
    MQ 消息确认与重试机制
    监控：
    Prometheus + Grafana，监控 MQ 消息积压、服务异常、延迟
    异常报警推送至钉钉/邮件
  7. 交付物清单
    全量代码实现：
    Controller、Service、Mapper、MQ 消费者
    数据库脚本：
    PGSQL 建表语句：K线表、告警表、模拟订单表、回测统计表
    部署套件：
    docker-compose.yml（包含 Consul, Apollo, RMQ, Postgres, Redis）
    Dockerfile 构建脚本
    deploy.sh 一键启动脚本
    README.md：
    项目介绍、启动指南、接口文档、回测说明
    前端可视化页面：
    实时行情、信号、模拟订单、回测分析与多参数对比
